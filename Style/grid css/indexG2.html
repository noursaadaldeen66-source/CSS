دليل شامل لخصائص CSS Grid

تعرف على جميع خصائص CSS Grid وكيفية استخدامها مع أمثلة تفاعلية

خصائص حاوية الشبكة (Grid Container)

هذه الخصائص تُطبق على العنصر الرئيسي الذي يصبح حاوية للشبكة.

display: grid
يحول العنصر إلى حاوية grid ويجعل جميع العناصر الفرعية عناصر grid.
.container {
  display: grid;
}
grid-template-columns
يحدد عدد الأعمدة وأحجامها في الشبكة.
.container {
  grid-template-columns: 100px 1fr 2fr;
}
grid-template-rows
يحدد عدد الصفوف وأحجامها في الشبكة.
.container {
  grid-template-rows: 50px 100px auto;
}
grid-template-areas
يحدد تخطيط الشبكة باستخدام أسماء المناطق.
.container {
  grid-template-areas:
    "header header header"
    "sidebar content content"
    "footer footer footer";
}
gap (row-gap, column-gap)
يحدد المسافة بين خلايا الشبكة (الصفوف والأعمدة).
.container {
  gap: 20px;
  /* أو */
  row-gap: 10px;
  column-gap: 20px;
}
justify-items
يحدد المحاذاة الأفقية لكل العناصر داخل الشبكة.
.container {
  justify-items: start | end | center | stretch;
}
align-items
يحدد المحاذاة الرأسية لكل العناصر داخل الشبكة.
.container {
  align-items: start | end | center | stretch;
}
place-items
اختصار لخصائص align-items و justify-items.
.container {
  place-items: center stretch;
}
justify-content
يحدد المحاذاة الأفقية للشبكة بأكملها داخل الحاوية.
.container {
  justify-content: start | end | center | stretch | space-around | space-between | space-evenly;
}
align-content
يحدد المحاذاة الرأسية للشبكة بأكملها داخل الحاوية.
.container {
  align-content: start | end | center | stretch | space-around | space-between | space-evenly;
}
place-content
اختصار لخصائص align-content و justify-content.
.container {
  place-content: center space-between;
}
grid-auto-columns
يحدد حجم الأعمدة التي تم إنشاؤها تلقائيًا.
.container {
  grid-auto-columns: minmax(100px, auto);
}
grid-auto-rows
يحدد حجم الصفوف التي تم إنشاؤها تلقائيًا.
.container {
  grid-auto-rows: 100px;
}
grid-auto-flow
يتحكم في كيفية وضع العناصر تلقائيًا في الشبكة.
.container {
  grid-auto-flow: row | column | row dense | column dense;
}

خصائص عناصر الشبكة (Grid Items)

هذه الخصائص تُطبق على العناصر الفرعية داخل حاوية الشبكة.

grid-column-start / grid-column-end
يحدد موقع بداية ونهاية العنصر بالنسبة لخطوط الأعمدة.
.item {
  grid-column-start: 1;
  grid-column-end: 3;
}
grid-row-start / grid-row-end
يحدد موقع بداية ونهاية العنصر بالنسبة لخطوط الصفوف.
.item {
  grid-row-start: 2;
  grid-row-end: 4;
}
grid-column
اختصار لخصائص grid-column-start و grid-column-end.
.item {
  grid-column: 1 / 3;
  /* أو */
  grid-column: 1 / span 2;
}
grid-row
اختصار لخصائص grid-row-start و grid-row-end.
.item {
  grid-row: 2 / 4;
  /* أو */
  grid-row: 2 / span 2;
}
grid-area
يعطي اسمًا للعنصر للإشارة إليه في grid-template-areas، أو كاختصار لخصائص تحديد الموقع.
.item {
  grid-area: header;
  /* أو */
  grid-area: 1 / 1 / 3 / 3;
}
justify-self
يحدد المحاذاة الأفقية لعنصر داخل خليته.
.item {
  justify-self: start | end | center | stretch;
}
align-self
يحدد المحاذاة الرأسية لعنصر داخل خليته.
.item {
  align-self: start | end | center | stretch;
}
place-self
اختصار لخصائص align-self و justify-self.
.item {
  place-self: center stretch;
}

وحدات القياس الخاصة بـ Grid

وحدة fr
تمثل جزءًا من المساحة المتاحة في حاوية الشبكة.
grid-template-columns: 1fr 2fr 1fr;
minmax()
يحدد نطاقًا للحجم بين minimum و maximum.
grid-template-columns: minmax(100px, 1fr) 2fr;
repeat()
يكرر نمطًا من الأحجام لعدد محدد من المرات.
grid-template-columns: repeat(3, 1fr);
auto-fit و auto-fill
تستخدم مع repeat() لإنشاء عدد متغير من المسارات حسب الحاجة.
grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

ملاحظة هامة

CSS Grid هو نظام تخطيط ثنائي الأبعاد تم تقديمه في CSS3، وهو مدعوم بشكل كامل في جميع المتصفحات الحديثة. يمكنك الجمع بين Grid و Flexbox لإنشاء تخطيطات معقدة ومتجاوبة.
