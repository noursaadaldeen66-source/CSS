شرح خاصية grid-column: 1 / -1

ما معنى grid-column: 1 / -1؟

هذه الخاصية تُستخدم لعنصر داخل حاوية Grid لجعله يمتد من أول خط عمودي إلى آخر خط عمودي في الشبكة.

  • 1: تشير إلى بداية العنصر عند أول خط عمودي (الخط رقم 1)
  • -1: تشير إلى نهاية العنصر عند آخر خط عمودي (الخط قبل الأخير + 1)
.element {
  grid-column: 1 / -1;
}

مثال عملي

في الشبكة أدناه، لدينا 6 عناصر عادية وعنصر واحد (رقم 5) يستخدم grid-column: 1 / -1 ليمتد على كامل العرض.

1
خط البداية
2
بين العمود 1 و 2
3
بين العمود 2 و 3
4
خط النهاية (-1)

بدون grid-column: 1 / -1

العناصر تأخذ مساحة عمود واحد افتراضيًا

.grid-item {
  /* لا يوجد grid-column محددة */
}

باستخدام grid-column: 1 / -1

العنصر يمتد على جميع الأعمدة

.grid-item {
  grid-column: 1 / -1;
}

متى نستخدم هذه الخاصية؟

تُستخدم grid-column: 1 / -1 في التصاميم التي تحتوي على:

  • رأس الصفحة أو تذييلها الذي يمتد على كامل العرض
  • أقسام خاصة داخل المحتوى تحتاج إلى تمييز
  • عناصر إعلانية أو رسائل مهمة
  • تصميم متجاوب حيث يحتاج العنصر إلى ocupying كامل العرض على شاشات صغيرة
/* مثال على استخدام عملي */
.header, .footer {
  grid-column: 1 / -1;
}

@media (max-width: 768px) {
  .sidebar {
    grid-column: 1 / -1;
  }
}
